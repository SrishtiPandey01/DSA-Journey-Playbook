📘 Chapter 01: Programming Basics for DSA

🚀 Overview

This chapter focuses on building a strong foundation in programming fundamentals and problem-solving using C++.

It includes basic mathematical problems, number manipulation, and logical thinking exercises that are essential before diving into core Data Structures & Algorithms.

---

🎯 Objectives

- Strengthen basic programming concepts
- Improve logical thinking and problem-solving skills
- Understand time complexity at a beginner level
- Build confidence in writing C++ programs
